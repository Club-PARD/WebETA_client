## 🎄 Project Structure

	🔥 client🔥
	├─public
	│ ├─index.html (기본 HTML)
	│ └─favicon.ico (파비콘)
	├─src
	│ ├─asset (이미지)
 	│ │ └─img (이미지)
	│ ├─components (컴포넌트)
 	│ │ ├─Auth (로그인)
  	│ │ ├─Home (홈)
	│ │ ├─Fandom (팬덤방)
 	│ │ └─data.js (데이터 like list)
   	│ ├─pages (페이지)
	│ │ ├─HomePage.js (홈페이지)
	│ │ ├─LoginPage.js (로그인 페이지)
	│ │ ├─MyPage.js (마이페이지)
	│ │ ├─SignupPage.js (회원가입 페이지)
	│ │ ├─CreateRoomPage.js (방 생성 페이지)
	│ │ ├─FandomRankingPage.js (랭킹 페이지)
	│ │ └─FandomRoomPage.js (팬덤방 페이지)
	│ ├─App.js (라우팅)
	│ └─index.js (root)
	├─package.json
 	├─.gitignore
  	├─Readme.md
 	└─package-lock.json
